std::int32_t okapi::DistanceSensor::getConfidence ( ) const

Get the confidence in the distance reading. This value has a range of [0, 63]. 63 means high confidence, lower values imply less confidence. Confidence is only available when distance is greater than 200 mm.

Returns
The confidence value in the range [0, 63].

std::int32_t okapi::DistanceSensor::getObjectSize ( ) const

Returns
