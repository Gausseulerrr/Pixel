\doxysubsection{Detailed Description}
A class representing a Proportional-\/\+Integral-\/\+Derivative (\mbox{\hyperlink{classpxl_1_1PID}{PID}}) controller. 

This class provides a convenient way to implement and use a \mbox{\hyperlink{classpxl_1_1PID}{PID}} controller for feedback systems.
