Prevents parenting the internal tasks started by this builder to the current task, meaning they will not be deleted once the current task is deleted. This can cause runaway tasks, but is sometimes the desired behavior (e.\+x., if you want to use this builder once in {\ttfamily autonomous} and then again in {\ttfamily opcontrol}).
